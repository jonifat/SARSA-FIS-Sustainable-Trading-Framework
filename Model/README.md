# Model Folder

This folder stores the trained model outputs from the SARSA-FIS trading framework.

## Files
- **Q_TABLE_SARSA_Training_Result.xlsx** – The learned state–action Q-table from SARSA reinforcement learning training.

## Description
The Q-table maps three market states (S1, S2, S3) and three possible actions (Sell, Hold, Buy).  
These values guide the agent’s decisions based on past rewards and expected outcomes.

## Note
This model output supports reproducibility of results but does not include full proprietary weights currently under patent review.
